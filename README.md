# E-commerce SQL Analytics

SQL analytics project built in Google BigQuery using an e-commerce dataset.

## Project Goals

- Analyze account creation trends
- Measure email engagement (sent, opened, and visited emails)
- Compare customer activity across countries
- Rank countries by account creation and email activity

## Technologies

- SQL
- Google BigQuery
- Looker Studio

## SQL Features Used

- CTE (Common Table Expressions)
- JOINs
- UNION ALL
- Aggregations (COUNT, SUM)
- Window Functions (DENSE_RANK, SUM OVER)
